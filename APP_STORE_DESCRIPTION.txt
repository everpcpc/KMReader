KMReader - Full-Featured Native Komga Client for iOS, macOS & tvOS

KMReader is a full-featured native SwiftUI client for Komga that brings your complete digital library to iPhone, iPad, Mac, and Apple TV. Read comics and books with powerful readers, download for offline access, browse with advanced filters, and manage your server — all without leaving the app.

KEY FEATURES

• Multi-Platform Support: Native experience across iOS, macOS, and tvOS with platform-specific optimizations.

• Advanced Readers: DIVINA reader with LTR/RTL/vertical/Webtoon modes, spreads, zoom, and page curl transitions. Native EPUB engine with custom fonts (.ttf/.otf), theme presets, and multi-column layouts. Per-book preferences save your reading settings.

• Offline Downloads: Background downloads with Live Activities on iOS. Series policies (manual, unread-only, all books) with automatic sync when reconnected. Three-tier caching keeps pages, books, and thumbnails available offline.

• Multi-Server Vault: Save unlimited Komga servers with password or API key authentication. Quick switching with isolated data per instance.

• Smart Dashboards: Dynamic sections (Keep Reading, On Deck, Recently Added) with real-time SSE updates. Advanced metadata filters with all/any logic for authors, genres, tags, and publishers.

• Admin Tools: Edit metadata, manage libraries, scan collections, monitor tasks, and view logs without leaving the app. Directory browser for library path selection.

• Privacy & Security: Incognito mode for private reading. Credentials stored securely on-device. Optional blur protection for reading content.

• Platform Features: Live Activities and background downloads on iOS. Separate reader windows and keyboard shortcuts on macOS. Remote control navigation on tvOS.

Whether you're reading on the couch or managing your server on the go, KMReader provides a complete full-featured native experience for your Komga library.
