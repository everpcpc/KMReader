New Features

- Replaced the reading engine with a custom native version for a more lightweight, stable, and customizable experience
- Added support for one or two column layouts in the EPUB reader
- You can now import your own fonts (.ttf/.otf) directly from the Files app
- New EPUB theme presets and an optimized theme selection menu
- Full support for nested and hierarchical Table of Contents
- Added a "Resume" button to series pages to quickly jump back into your reading
- Enhanced reader controls with new gradient background options and a streamlined interface
- Improved navigation with gesture-based chapter switching in scroll mode
- Multi-language preview for custom fonts to help you choose the best look

Improvements

- More accurate reading progress tracking for EPUB books
- Faster and more reliable page loading and layout rendering
- Better accessibility with larger control buttons and improved label visibility
- Redesigned settings menu for easier navigation and better organization
- Enhanced offline download management and more accurate file size reporting
- Specific UI optimizations for a better experience on macOS
- Upgraded core system for improved overall app performance

Bug Fixes

- Fixed an issue where the reader might snap back during the first page turn
- Resolved issues with zoom transitions and page animations
- Fixed custom font previews and improved font rendering across all book content
- Smoother page turns with improved tap zone logic and gesture handling
- Fixed a bug that prevented re-downloading books that were already marked as read
- Improved network reliability and timeout handling
- Fixed various styling issues with images, tables, and CSS in EPUB books
