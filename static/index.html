FAQ

Is KMReader an official Komga app?

No. KMReader is an open-source community client that uses the public Komga APIs. It is developed independently from the Komga project.

Can I connect to multiple Komga servers?

Yes. Save unlimited servers with custom names using password or API key authentication. Switch instantly between servers with isolated data per instance.

What formats can I read?

DIVINA comics work on all platforms (iOS, macOS, tvOS) with every reading direction. EPUB books are available on iOS and macOS with a native engine. tvOS currently supports DIVINA only.

How does offline reading work?

Download books with background downloads (iOS) or foreground downloads (macOS/tvOS). Set series policies to automatically download unread books. Three-tier caching stores pages, books, and thumbnails with adjustable limits.

Can I manage my Komga server?

Yes. Admin users can edit metadata, create and scan libraries, manage collections and read lists, monitor tasks, and view logs directly in the app.

What Komga version is required?

KMReader requires Komga 1.19.0 or later. It works with Komga API v1 and v2 and supports SSE for real-time updates.
